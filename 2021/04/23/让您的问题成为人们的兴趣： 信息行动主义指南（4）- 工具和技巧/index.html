活动家的信息行动如何才能是有效的
有效的数字媒体信息行动是指那些旨在吸引目标受众或目标社区的注意力，激起或激励他们的活动。

📌 一般来说，这些活动都是在能够最大范围地接触到人们的平台上进行的，或者劫持现有的普遍性平台，如广告或流行文化的出口。它们可以由较小的群体策划，但却能影响到更大的群体，虽然不一定直接激起目标群体的行动，而是更注重唤醒人们对一个被隐藏的严重问题的关注和思考。

通过创造力和幽默感引起人们对某个问题的关注、或绕过审查；

使用久经考验的基于打印的技术离线创建影响力；

让您的信息在大众媒体的海洋中脱颖而出。

干扰信号：创造性信息行动

从根本上说它的确是战术，这一战术可被任何人使用，您的对手一直在使用，但追求程序正义的活动家可能很难想象，事实上反抗者也可以使用这一战术。

这就是为什么我们将战略效用的讲述放在第2集中，您应该先了解战略和战术的定义及策划技巧，就更容易理解本系列后面将要讲述的话题了。

雨林行动网络（RAN）、亚马逊观察（AW）和 Yes Men 这些行动组织开始通过一个适时的文化干扰技巧来破坏雪佛龙的 “洗绿”。

📌 活动家们计划从一个假冒雪佛龙域名的恶搞新闻稿开始，他们在真正的雪佛龙发起实际宣传活动之前的几个小时内推出了一个假冒的 “我们同意” 网站。这个虚假的 “我们同意” 网站上有四条 “改进” 的广告，其中包括 “石油公司应该清理他们的烂摊子” 和 “石油公司应该尊重民主制度，而不是操纵民主制度”，等环保行动口号。

当雪佛龙发布自己的广告时，活动家们制作的这个假广告已经在不同的媒体渠道发布了。当雪佛龙试图补救时，活动家们继续发布更多的虚假信息，让公众感到困惑。这迫使该公司发布声明，“揭露” 另一个运动是假的，从而吸引了公众更多的注意力，当然，人们也因此更多看到了该公司有争议的商业和环境实践。

文化干扰
📌 文化干扰（Culture jamming）指的是挑战或破坏主流语话的行动主义，带有一定的颠覆性和创造性 — — 使用黑客攻击和传播假的信息作为战术，请注意，所有这些战术都必须满足和围绕中心战略。

目的在于反抗主流媒体及其塑造的主流文化，并暴露商业文化可能存在的问题、重新塑造文化定义。

常用手段为异轨  — — 异轨（法语：détournement）是当代媒体作品的变体，它是一种 “将旧有作品以颠倒的方式重新创作” 的手法，而被选做重新创作的原作品（détourned）必须是一个被大众所熟悉的媒体，以便能够有效和迅速的传达与原作相反的意图和信息。异轨类似一种有讽刺意味的模仿，不过它是采取直接引用或忠实模拟原始作品来制造这种强烈的暗示，而不是重新创作新的作品来达到同样的效果。以此来传达相反的意念、颠覆主流媒体及传递自己的作品。

“Culture jamming” 一词被认为由美国 “ Negativland” 乐队的唐·乔伊斯（Don Joyce）在1984年提出，但实际上在1950年代这个词就已经出现了。

📌 为了制造混乱，文化干扰行动者经常使用主流媒体渠道和营销人员使用的相同工具。颠覆（Subvertisments） — — 巧妙地颠覆广告 — — 就是这样一个例子。希贾布公主在巴黎地铁中 “niqab-ising” 衣着光鲜的广告画面，让自己名声大噪。

Adbusters 是另一个著名的资源，它是一本以颠覆 Diesel、Calvin Klein 和 Nike 等著名品牌为主题的杂志。这样的例子可以追溯到 Ester Hernández 等艺术家的作品，她在80年代初就已经开始创作颠覆广告，特别是她的 Sun Mad 海报，在加州被查封，因为她提请人们注意农药使用对环境的污染。

【注：埃斯特·埃尔南德斯（EsterHernández）是旧金山的 Chicana 视觉艺术家，她的作品包含政治、社会、生态和精神主题，是社区行动和政治行动。】

为什么要使用这种方法？
作为活动家，当您想要引起人们对某个特定团体或公司的不道德行为的关注时，文化干扰是一种极好的策略。

📌 尤其是，对于资源紧张、无法开展反击活动的组织来说，文化干扰是一种可行的选择。由于广告的目的都是为了接触到大量的受众，于是有效地劫持这个平台也会起到同样的作用。

在您的信息行动中使用幽默元素也会更吸引人，使信息更难忘，就像一丝意外的暗示一样。诸如篡改产品贴纸或标签并将其送回商店货架上去的做法，正是因为出人意料和直接，才会产生强烈影响。比如看这里 shop-dropping。

如何在您的行动中运用这一战术？
以下是在运用文化干扰技巧时您需要注意的一些一般提示：

1、平衡信息与艺术
上面所举的例子，虽然在视觉上和概念上都很吸引人，但也有明确的、坚实的信息支持。不要让容器掩盖了您真正想呈现给受众的东西。确保风格与内容相辅相成，而不是相反。

2、了解您的受众
正如每个广告牌都有目标人群一样，您的信息行动也应该有特定的受众。试着想一想您的受众群体最喜欢的媒体平台、语言、地理位置和其他因素，这些因素会影响您的行动的覆盖面。

例如，“Occupy George” 活动中，活动家在美元纸币上印上了关于财富不平等的揭示性信息，并向美国公民发放。由于真正的美元纸币很容易进入流通领域，这就让活动家的信息可以传得很远。

3、时机就是一切
不要低估时机成熟的行动的效果。在某些情况下，您可以将行动的时间安排在已经计划好的活动或纪念日上。在其他情况下，最好在突发事件发生后迅速做出反应  — — 劫持由此产生的媒体焦点。

4、了解您的文化参考
您对一个地方或地区的文化货币越熟悉，您就越能最有效地颠覆、翻转或破坏它。一个文化干扰行动如果脱离了原来的语境，就会变得失去意义。

5、要有说服力
文化干扰行动不能半途而废。如果您能设法干预官方的交流渠道，您就就能成功地引起公众和媒体的注意。例如，“Yes Men” — — 他们冒充大公司首席执行官或发布虚假新闻稿进行恶作剧  — — 就是这方面的高手。

法律问题
篡改标识或冒充企业可能会产生严重的法律后果，您应该在策划您的活动之前了解这些后果。企业材料受所谓的知识产权保护，将这些材料用于反宣传活动可能会被 “诽谤” 倒打一耙。

最常见的变通方法是在合理使用的情况下申请；如果您不确定这点，请寻求当地的法律咨询。

【注：合理使用，是著作权法的一个概念，允许人们于某些情况下，无需征求著作权所有者的同意，就可以使用受著作权保护的部分内容。“合理使用”这个概念在美国等许多国家的著作权法中都存在。合理使用试图在著作权所有人的利益和公众利益之间取得平衡，在兼顾原创者利益的同时又鼓励新的创造。】

案例探究
在世界各地，对政客和政府或国家名人感到愤怒  — — 甚至只是感到困惑的网民，一直在利用嘲弄、讽刺和荒诞作为表达异议和捣毁权力的措施。从朝鲜总统看东西的照片到希拉里·克林顿meme，Tumblr 博客经常这样引来笑声，并对政治、社会和文化进行讽刺性的洞察。

ZA News 等恶搞新闻网站通过对时事的艺术模仿来吸引读者的注意力，用幽默和讽刺的方式来切入官方新闻，揭露官方新闻的荒诞本质。Molleindustria 创造的游戏揭示了技术所依赖的机构和行业的黑暗面。

📌 恶作剧网站可以实现狠狠地打击  — — 例如假阿萨德基金会网站  — — 实际上是对叙利亚政权暴行的尖锐批判  — — 以及卢卡申科的 Lu-Net，这是一群白罗斯活动家为了嘲讽亚历山大·卢卡申科总统而建立的网站，因为他承诺要加强网络审查。

视频混音可以是一种围绕有限的视频资源的创造性方式。利用在互联网上找到的现有镜头和一些编辑软件，视频混音有可能实现病毒式传播，以覆盖大量观众。Zenga Zenga 将穆阿迈尔·卡扎菲的演讲片段与嘻哈节奏进行了混搭；A Fair(y) Use Tale 将迪斯尼电影的片段拼接在一起，以说明知识产权的荒谬性。

如果CC许可的内容不适合您的需要，您可以匿名发布您的影片。通常情况下，改编的作品会受益于版权法中的 “公平使用” 或 “公平交易” 条款。

实践：创建混搭
📌 由于混搭的性质，任何存在的规则都是要被打破的。这是一个好主意，玩转现有的惯例，使您的作品引人注目。以下是一些惯例：

使用器乐音轨作为音频基础，音轨要适合您想要传达的信息；

选择一系列的采访，说明您想表达的观点，并将采访的基调与您的项目的整体感觉相匹配；

抓取流行的片段 — — 花絮、新闻报道、幕后花絮 — — 与严肃片段形成强烈反差；

找出与主题相关的背景视频，或作为一种创造性的对比；去掉它的原有音频，并将其与其他视频穿插在一起。

印刷媒体：远未过时
虽然印刷媒体被认为是数字媒体的前身，但它如今依旧非常有效。它的简单性和即时性使它即使在今天也是一个宝贵的工具。

在与技术隔绝的社区中，印刷媒体影响变化的潜力是不言而喻的；但即使在技术丰富的社区中，印刷媒体作为数字媒体运动的补充或替代，也是非常有效的。软件方面的进步使预算紧张的活动家比以往任何时候都更容易制作和分发印刷媒体（使用数字手段）。

为什么要使用它？
印刷品并没有失去其激励和刺激行动的能力，但它的力量在网络文字的海洋中被冲淡了。使用印刷品媒体是一个让您的活动回归街头的机会。印刷品传播可以从根本上放大你的信息，而且往往只需花费很少的成本和精力。

一件印有图案的T恤衫，在正确的时间、正确的地点穿上，或者是醒目的贴纸，都可以发出大胆的声明。以您的主要受众为对象制作的月度通讯，可以成为邀请基层社区参与的绝佳方式。其他形式包括海报、传单、杂志、贴纸和小册子等实物分发。

🔧 工具1：Scribus
Scribus 是一款页面布局程序。它可以与 Microsoft Publisher 或 Adobe InDesign 相比。

使用 Scribus，您可以为通讯、公告、手册、报告和其他需要文本和图像布局的印刷材料创建布局。当您要创建更专业的出版物时，它可以是一个很好的工具。

Scribus 具有友好的用户界面，但也支持高级功能，如分色、CMYK 和灵活创建 PDF 文档。

您可以用它来做什么：

制作宣传和活动传单、宣传单和海报，注重视觉的精确性。

用书本和小册子交流研究资料、经验和分析。

优点：

可以超越基于Office的排版限制，采用更专业的印刷方式。

与 LibreOffice 良好的兼容性 — — 可以导入格式和样式。

缺点：

与文字处理程序相比，创建文档更难上手。

它不能与其他桌面文件格式（如 InDesign 或 Publisher）一起工作。

平台(操作系统)：

Windows, Mac OSX, Linux

培训材料：http://wiki.scribus.net/canvas/Help:Manual_Intro

替代方案：

Microsoft Publisher

Adobe Indesign

LibreOffice

🔧 工具2：LibreOffice
LibreOffice Writer 是一款文字处理和桌面出版工具。它可以用于简单的任务，但也能够创建传单、小册子甚至书籍。

它是一款名为 LibreOffice 的应用程序系列的一部分，它包含了创建图表、电子表格和演示文稿的工具，这些工具可用于您的印刷工作。它允许使用样式表，您可以借此创建一致的布局。

它支持以开放的文档格式和许多不同版本的 Microsoft Word 打开和保存。您还可以将您的作品保存为PDF文件，这对于打印文件来说非常方便。

您可以用它来做什么：

撰写较长的文字，以多种不同的形式与队友分享。

制作海报、贴纸和传单来传达您的信息。

由于使用了模板友好型字体，可以制作出可以作为模板使用的喷漆打印稿。

优点：

易于使用，并提供了良好的帮助

可以处理和导出多种不同的文件格式。

缺点：

文件以专有格式保存，可能无法在所有系统上读取

平台(操作系统)：

Windows, Mac OSX, Linux

培训材料：http://www.libreoffice.org/get-help/documentation/

🔧 工具3：Gimp
GIMP（GNU Image Manipulation Program）几乎可以做到 Adobe Photoshop 能做到的所有事，而且任何人都可以免费使用。

您可以使用GIMP来调整照片或其他图像的大小和裁剪，以及调整图像的亮度和对比度，为图形添加文字，应用各种效果和滤镜，使用图层，创建动画GIF，并让图像准备好打印或上网。

您还可以编辑或删除照片中包含的元数据。只需在保存图像时，在高级选项中取消勾选 “保存EXIF数据” 框即可。

您可以用它来做什么：

编辑照片用于您的信息行动主义活动博客或网站。

通过删除照片中的EXIF元数据，防止您或您的队友被识别。

改变任何图像文件，用于海报、传单或其他信息艺术作品。

优点：

免费软件，可用于许多操作系统

与 Photoshop 非常相似，易于使用

缺点：

每张图片都会出现在自己的窗口中，除非您选择 “单窗口模式”，这对第一次使用的用户来说可能会有点困惑。

平台(操作系统)：

Windows, Mac OSX, Linux

培训材料：http://www.gimp.org/docs/

策划方法
在最初的策划阶段，您要考虑以下因素：

受众 — — 社会群体、政治倾向、地理位置、年龄；

风格  — — 语言、语气、正式程度；

成本  — — 可持续性、印刷成本、纸张和储存成本；

发行  — — 方法、地理分布、份数；

贡献者  — — 是否有积极和有能力的贡献者，或针对性培训。

这些问题将极大地影响您的内容创作过程，并最终影响您对格式的选择。

选择正确的格式也是为了确保您的项目与您的整体传播战略有明确的联系。在大多数情况下，以印刷为基础的媒体，其输出结果是无法像网站那样进行修改的，因此，为了避免代价高昂的错误，仔细规划是必不可少的。

📌 在决定选择哪种形式时，请考虑您的目标、能力和受众。您是想让人们在一个问题上快速行动，还是您有一个本身就很有意义、不会很快过时的信息？简单的信息可以采用较小的一次性格式，如海报、贴纸和 T恤衫，或者是小册子或传单，其中可以包括链接到您的网站上更频繁的更新。海报或贴纸可以由志愿者张贴，传单可以在示威活动中分发。

如果您想通过让人们深入了解您的事业所涉及的问题来长期吸引他们，或者如果您的信息一直在变化，并且与其他发展相关，那么最好的办法是出版通讯或杂志。

为了突出某个特定的主题，您也可以发行一次性的出版物，如简报或公报  — — 但不要告诉你的听众它将成为一个系列的一部分，除非您已经承诺要制作更多的刊物！

一个较长的文本可能需要作为报告，甚至作为一本书出版。如果您的预算较少，它可以成为电子书供下载。请记住，任何在线出版的东西都可以发行更定期、更便宜、更容易地更新。

使用各种格式的组合是在短期和长期内达到所有目标和受众的最佳方式。

2009年，为了在沙特公民中引发关于当局禁止女性驾驶的社会对话，一位居住在美国的沙特艺术家和平面设计师发起了一场名为 “我们是女人” 的活动。Areej Khan 设计了一套 “宣言泡泡” 贴纸，鼓励人们打印出来，填上自己的想法，然后在公共场所分发，包括在汽车上。支持者还被鼓励拍摄他们的个性化贴纸，并上传到该行动主义项目的 Flickr 集或 Facebook 页面上，或者以匿名方式将贴纸发到 Areej Khan 的邮箱。

巴勒斯坦艺术家 Khaled Jarrar 开发了一枚 “巴勒斯坦国” 邮票，他用这枚邮票在加沙游客的护照上盖章，以提高人们对该领土仍无权建立自己的边境管制这一事实的认识。

在印度，“零元卢比纸币 Zero Rupee Note” 是 Vijay Anand 通过他的组织 “第五支柱” 创造的一个运动工具。零元卢比纸币是一种视觉辅助工具，用于动员人们对腐败官员说 “不”，并揭露索贿的地点和时间。它的设计以印度50卢比纸币为基础，但纸币上印有英语和泰米尔语的信息  — — “消除各级腐败” 和 “我保证不受贿也不行贿”。

在瑞士，活动家组织 “脏手” 推出了一个可打印的迷你尖塔，以抗议当地2009年通过的修正案，禁止建造尖塔。用户可以下载一个PDF文件，将迷你尖塔打印到纸板上，用胶水粘起来，然后展示在阳台或窗户上，以抗议法律的荒谬。

@Verdade 是一份独立的报纸，于2008年在莫桑比克推出，提供平衡的新闻，并针对那些生活在非正式城市定居点的贫困人口。该报迅速成为一个更大的大众运动的一部分，它激励了曾经被局限于国家广播和电视的人口。

该报完全免费提供（这也是该报的基本原则之一），通过广告收入和其他创新手段，如出租送报车辆，该报得以保持可持续性。

虽然该报有一个平行的网络版，但在一个只有不到10%的人口能用上电的国家里，将印刷版报纸作为重点是非常重要的。@Verdade 是一个很好的例子，说明在这个所谓的数字时代，印刷型媒体仍然可以发挥重要作用。
